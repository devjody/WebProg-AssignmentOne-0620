Hello,

Welcome to my school project for Intro to Web Programming!

Here you will find a few Javascript coded pages with some features like an input driven 'Mark To Grade Converter', an Object/Array based 'Data Sorting Staff List' and finally a simple 'Tempereture Conversion Tool' from Fahrenheit to either Celsius or Kelvin!

As you would expect this website is using the usual HTML/CSS, however during this build I tested many CSS frameworks out like Bootstrap, PaperCSS and Materialize! In the end, I decided to give 'simplegrid.io' a shot and its fairly handy to work with.

For many popular frameworks, there is an option to run a CDN link. However, this project has all the core files included in the package and then some CSS to fine tune it to my design. Also the artwork used is my own too!

Many frameworks I have tried so far use either flexbox grid style model and mobile first idealogy, where the div's have unique css properties attached to them, for example...

<div class='container'>
<div class='row'>
<div class='col'>

Once you understand the class system in each framework, it allows you to quickly format your document to responsive sizes fairly quickly. However one draw back is if you wish to make a fully customised website, you may find yourself ripping apart the framework to override the defaults.
